# Cosmic Explorer - Solar System Guide

A stunning, responsive website dedicated to exploring our Solar System. Built with modern web technologies and featuring a dark space theme, this educational platform provides comprehensive information about all eight planets.

## Features

- **9 Interconnected Pages**: Home page plus dedicated pages for Mercury, Venus, Earth, Mars, Jupiter, Saturn, Uranus, and Neptune
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices
- **Dark Space Theme**: Immersive dark theme with glassmorphism effects and smooth animations
- **Mobile Sidebar Navigation**: Elegant slide-out navigation menu for mobile users
- **Rich Content**: Each planet page contains 1000+ words of detailed information
- **Modern UI/UX**: Built with Tailwind CSS featuring custom space-themed color palette
- **Smooth Animations**: Hover effects, transitions, and interactive elements throughout
- **Typography**: Premium fonts from Google Fonts (Inter & Bellefair)

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No build tools or dependencies required!

### Installation

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start exploring the solar system!

```bash
# If using a local server (optional)
# Python 3
python -m http.server 8000

# Node.js
npx serve
```

Then navigate to `http://localhost:8000` in your browser.
